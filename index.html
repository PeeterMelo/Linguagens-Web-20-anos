Linguagens de Programação WEB no futuro

Como serão as Linguagens de Desenvolvimento Web daqui 20 anos

Acreditamos que entre esses anos as linguagens já usadas atualmente para desenvolvimento  web serão apropriadas e modificadas por empresas que que compraram ou fizeram uma versão muito diferente da mesma, assim como vemos no decorrer da história do tão usado e popular JavaScript

O JavaScript é um bom exemplo para fazer esta previsão em especifíca porque sofreu diversas mudanças ao decorrer do tempo de diferentes empresas desde que foi criado junto ao Java.

Sabendo disto, é plausível falar que com certeza novas linguagens como o Javascript, Phyton e C receberão modificações inovadoras e/ou até mesmo a criação de uma nova linguagem a partir da concorrência e necessidade da mesma no mercado de trabalho.
